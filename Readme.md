# Feature Request Application README

##Technology stack
- PostgreSQL
- Python 2.x
- Flask
- SQLAlchemy

##Installation

```
# Step 1: install virtualenv
> pip install virtualenv

# Step 2: Initialize virtualenv
> virtualenv <name-optional>

# Step 3: Activate virtualenv
> source bin/activate

# Step 4: Setup project
> python setup.py
```
